## SAM-10 GNB 시안 제안서

작성일: 2026-03-13
대상: netpple.github.io 상단 GNB 2차 개선

## 공통 목표
- GNB의 미완성 인상을 줄이고 브랜드 일관성을 강화
- 현재 위치(active) 인지성을 높여 섹션 이동 비용을 낮춤
- 데스크톱/모바일 모두 동일한 상호작용 규칙(hover/active/toggle)을 유지

## 시안 A: 컴팩트형(미니멀 바)
- 의도: 상단 높이를 최소화해 본문 집중도를 극대화
- 데스크톱 동작
  - Hover: 텍스트 컬러만 강조
  - Active: 하단 2px 인디케이터 표시
- 모바일 동작
  - Toggle: 햄버거 클릭 시 전체 너비 드롭다운
  - Active: 좌측 보더 강조
- 장점: 구현 리스크가 낮고 밀도가 가벼움
- 단점: 브랜드 존재감이 상대적으로 약함

## 시안 B: 브랜드 강조형(배지 + 텍스트 메뉴)
- 의도: 로고 배지와 브랜드명을 가장 강하게 노출
- 데스크톱 동작
  - Hover: 메뉴 캡슐 배경 강조
  - Active: 브랜드 컬러 배경 고정
- 모바일 동작
  - Toggle: 브랜드 아래 카드형 메뉴 확장
  - Active: 카드 내부 강조 배경 유지
- 장점: 첫 인상/기억성 강화
- 단점: 시각 밀도가 높아져 본문 대비가 약해질 수 있음

## 시안 C: 세그먼트형(선택안)
- 의도: 메뉴를 하나의 세그먼트 그룹으로 묶어 정보 구조를 직관화
- 데스크톱 동작
  - Hover: pill 단위 배경/미세한 상승 변환
  - Active: 흰색 pill 고정 + 진한 텍스트로 현재 섹션 강조
- 모바일 동작
  - Toggle: 햄버거 클릭 시 라운드 패널 확장 + 세로 스택
  - Active: 데스크톱과 동일한 강조 규칙 유지
- 장점: 섹션 전환 인지성이 높고 규칙 일관성이 좋음
- 단점: 컴팩트형 대비 커스텀 스타일 범위가 넓음

## 선택 결과
- 선택안: **시안 C (세그먼트형)**
- 선택 근거
  - Home/News/Docs/About 간 현재 위치 인지가 가장 명확함
  - hover/active/toggle 규칙이 단일 패턴으로 통일되어 학습 비용이 낮음
  - SAM-10 목표(브랜딩 정돈 + 사용성 개선)에 가장 직접적으로 부합
